# NPC 지식베이스 및 RAG 테스트

## 목적
NPC가 온보딩 문서 기반으로 질문에 답변할 수 있도록 지식베이스를 구축하고, RAG(Retrieval-Augmented Generation) 시스템을 구현합니다.

## 아키텍처

### 정형데이터 조회
직원 수, 부서 정보 등 구조화된 데이터는 TAG를 통해 데이터베이스에서 직접 조회합니다.

### 문서 기반 질문
온보딩 가이드 등 문서 기반 질문은 RAG 파이프라인으로 처리합니다.
