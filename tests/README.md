# 테스트 및 프로토타입 개발

이 폴더는 CG Town 프로젝트에서 평가 중인 기능들의 프로토타입 및 테스트 구현을 포함합니다.

## 폴더 구조

### meeting-room-booking/
회의실 자동예약 기능 프로토타입. 회의실 예약 시스템과의 API 연동 및 자동 예약 로직을 개발합니다.

### cafeteria-menu-scraper/
오늘 식당 메뉴 스크래핑. 식당 웹사이트에서 일일 메뉴 정보를 수집하고 NPC 말풍선으로 제공합니다.

### npc-knowledge-base/
NPC 지식베이스 구축 및 RAG(Retrieval-Augmented Generation) 테스트. 온보딩 문서 기반 질문 응답 및 정형데이터 조회 기능을 개발합니다.
